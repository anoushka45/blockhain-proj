Smart Contract Vulnerability Analyzer

The Smart Contract Vulnerability Analyzer is a tool designed to help developers identify and analyze vulnerabilities in their smart contract code. Supporting both Solidity (.sol) and Rust (.rs) file types, this tool scans contracts for potential issues and provides insights on common security risks, helping developers ensure their contracts are secure before deployment.
